GhostDesktop

v0.9.4 - Windows Utility

Now You See It,
Now You Don't.

Desktop icon fades smoothly from fully visible to fully hidden while your workflow stays uninterrupted.

GhostDesktop is a lightweight utility for Windows that dynamically fades desktop icons based on focus or hover, including complete icon disappearance. Keep your desktop clean, minimal, and instantly accessible when needed.

Everything you need,
right on your desktop.

Built to disappear when you do not need it, then return instantly when you do.

Smooth Fade Engine

Desktop icons transition in and out with tunable speed and delay values for a polished, subtle effect.

Focus or Hover Trigger

Choose click/focus-based behavior or hover-trigger mode from Settings to match how you use your desktop.

Persistent Settings

Your preferences are stored in `%APPDATA%\\ghostdesktop\\ghostdesktop.ini` and load automatically on startup.

Tray Menu Control

Quick access to Settings and About from the tray menu keeps configuration one click away.

Performance Tunable

Use high-speed, low-delay profiles for snappy transitions or slower curves for ultra-smooth visual fading.

Minimal Desktop Mode

Keep wallpaper and workspace in focus while icons gracefully vanish until you need to interact with them.
